# Boss Yu Team - 案件管理系统

律师事务所投融资团队案件管理系统

## 功能特点

- 📊 **工作台**：项目概览、我的任务、即将到期提醒、最新动态
- 📁 **项目管理**：投融资/并购/尽调/常法等类型，自动生成进度阶段
- ✅ **任务管理**：任务创建、分配、截止日期、状态跟踪
- 📅 **日历视图**：项目和任务时间安排一目了然
- ⏱️ **工时记录**：实时计时器 + 手动录入
- 👥 **团队管理**：成员管理、角色权限

## 使用说明

1. 使用预设账号登录
2. 创建项目，系统自动生成进度阶段
3. 在项目下添加任务
4. 使用计时器或手动录入工时

## 部署方式

推荐使用 Vercel 免费部署

## 技术栈

- 原生 HTML + Tailwind CSS + JavaScript
- 本地存储 (localStorage)
- 可选 Supabase 云数据库
